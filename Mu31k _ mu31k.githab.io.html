Начните с создания нового файла или загрузки существующего файла . Мы рекомендуем каждое хранилище включает в README , ЛИЦЕНЗИИ , и .gitignore .

… Или создайте новый репозиторий в командной строке

echo "# mu31k.githab.io" >> README.md 
git init 
git add README.md 
git commit -m "первая фиксация" 
git branch -M master 
git remote add origin https://github.com/Mu31k/mu31k. githab.io.git
 git push -u origin master

… Или нажмите существующий репозиторий из командной строки

git remote добавить источник https://github.com/Mu31k/mu31k.githab.io.git
 git branch -M master 
git push -u origin master
